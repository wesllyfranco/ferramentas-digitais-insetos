Tutorial: Crie seu Próprio Jogo da Memória - Insetos

Olá, futuro programador! Bem-vindo ao seu projeto de programação.

Vamos construir, passo a passo, um "Jogo da Memória" com o tema de insetos. O objetivo é encontrar os pares corretos, que neste caso são o nome popular do inseto e sua classificação científica (Ordem).

Este tutorial foi feito para quem está começando, então vamos explicar cada detalhe!

Parte 0: As Ferramentas e a Preparação

Antes de construir uma casa, você precisa de ferramentas e de um terreno. Na programação, é a mesma coisa.

1. A Ferramenta (O Editor de Texto)

  • Precisamos de um programa para escrever nosso código.
  • Recomendamos um programa gratuito chamado Visual Studio Code (ou "VS Code"). Ele é o mais usado no mundo, é fácil de usar e ajuda você colorindo o código.
  • Ação: Peça ajuda para baixar e instalar o "Visual Studio Code" no seu computador. Você pode baixar aqui. Há também uma versão online que pode ser acessada por esse link

2. O Terreno (A Pasta do Projeto)

  • Não podemos deixar nossos arquivos jogados na Área de Trabalho. Vamos criar uma "pasta" para guardar todos os arquivos do nosso jogo.
  • Ação:
    1. Vá para sua Área de Trabalho ou para a pasta "Documentos".
    2. Clique com o botão direito do mouse e escolha Novo > Pasta.
    3. Dê a esta pasta o nome de jogo-da-memoria.
    4. Abra o Visual Studio Code. No menu de cima, clique em File (Arquivo) > Open Folder... (Abrir Pasta...) e escolha a pasta jogo-da-memoria que você acabou de criar.

Tudo pronto! Estamos na nossa pasta e prontos para criar nossos 3 arquivos.

Parte 1: O Esqueleto do Jogo (HTML)

1. Crie o arquivo

  • Dentro do VS Code, na lateral esquerda (onde está o nome jogo-da-memoria), clique no ícone de "Novo Arquivo" (parece uma folha de papel).
  • Dê o nome de index.html e aperte Enter. É muito importante que termine com .html.

2. O Código (Copie e Cole)

Copie e cole o código abaixo no seu arquivo index.html.

index.html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Jogo da Memória - Insetos</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <h1>Jogo da Memória: Insetos</h1>
  
  <div id="game-board"></div>
  
  <p id="status"></p>

  <script src="script.js"></script>
</body>
</html>

3. Entendendo o Código HTML

O HTML é a estrutura, os "ossos" da nossa página. O que acabamos de colar faz o seguinte:

  • <!DOCTYPE html>: Diz ao navegador que este é um arquivo HTML moderno.
  • <head>...</head>: É a "cabeça" do site, com informações que não aparecem na tela, como o título da aba (<title>).
  • <link rel="stylesheet" href="style.css">: Esta é a linha mais importante da "cabeça". Ela "liga" este arquivo HTML ao nosso arquivo de estilos (CSS) que vamos criar a seguir.
  • <body>...</body>: É o "corpo" do site, onde todo o conteúdo visível fica.
  • <h1>...</h1>: Cria o título principal na tela.
  • <div id="game-board"></div>: Esta é a "caixa" (div) mais importante. Demos a ela o nome (id) de "game-board". Ela começa vazia, e o JavaScript vai usá-la para criar os cartões dentro dela.
  • <p id="status"></p>: Um parágrafo (<p>) vazio que usaremos para mostrar mensagens ao jogador, como "Você acertou!" ou "Tente novamente!".
  • <script src="script.js"></script>: Esta linha "liga" o HTML ao nosso arquivo de JavaScript (o "cérebro"). É crucial que ela esteja no final do <body>, para garantir que o resto do HTML seja carregado *antes* do JavaScript tentar manipulá-lo.

Parte 2: A Aparência do Jogo (CSS)

1. Crie o arquivo

  • Na mesma pasta, crie um novo arquivo.
  • Dê o nome de style.css e aperte Enter.

2. O Código (Copie e Cole)

Copie e cole o código abaixo no seu arquivo style.css.

style.css
body {
  font-family: Arial, sans-serif;
  text-align: center;
  background-color: #f5f5f5;
}
h1 {
  margin-top: 20px;
  color: #333;
}

#game-board {
  display: grid;
  grid-template-columns: repeat(4, 120px); 
  gap: 15px;
  justify-content: center;
  margin: 30px auto;
  max-width: 600px;
}

.card {
  width: 120px;
  height: 120px;
  background-color: #1c6dd0;
  border-radius: 8px;
  cursor: pointer;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 14px;
  font-weight: bold;
}

.card.flipped {
  background-color: #fff;
  color: #000;
  border: 2px solid #1c6dd0;
}

3. Entendendo o Código CSS

O CSS cuida da "decoração" (cores, tamanhos, layout). Ele funciona selecionando os elementos do HTML e aplicando estilos a eles.

  • body e h1: Estilizam o corpo da página (fonte, cor de fundo) e o título (cor, margem).
  • #game-board: O # significa que estamos selecionando um elemento pelo seu id.
    • display: grid;: Transforma nosso tabuleiro em um "grid" (uma grade).
    • grid-template-columns: repeat(4, 120px);: Esta é a regra principal do layout. Ela diz ao grid: "Crie 4 colunas, e cada uma deve ter 120 pixels de largura".
    • gap: 15px;: Cria um espaço de 15 pixels entre cada cartão no grid.
    • justify-content: center;: Centraliza o grid (o conjunto de cartões) no meio da página.
  • .card: O . significa que estamos selecionando elementos pela sua "classe" (class). Esta regra define a aparência do cartão "virado para baixo" (o verso).
    • width: 120px; height: 120px;: Define o tamanho do cartão, combinando com o tamanho da coluna que definimos no grid.
    • background-color: #1c6dd0;: Define a cor de fundo azul.
    • cursor: pointer;: Faz o mouse virar uma "mãozinha" quando passa sobre o cartão, indicando que é clicável.
    • display: flex; ...: Este conjunto de regras é um truque para centralizar perfeitamente o texto (o "?") dentro do cartão.
  • .card.flipped: Esta é a "mágica" do CSS. É um seletor que só se aplica a um elemento que tenha AMBAS as classes, .card e .flipped.
    • O JavaScript vai adicionar a classe .flipped quando clicarmos. O CSS, vendo isso, vai aplicar este estilo, trocando a cor de fundo para branco (#fff), a cor do texto para preto (#000) e adicionando uma borda azul. É isso que faz o cartão "virar" visualmente.

Parte 3: O Cérebro do Jogo (JavaScript)

1. Crie o arquivo

  • Na mesma pasta, crie o último arquivo.
  • Dê o nome de script.js e aperte Enter.

2. O Código (Copie e Cole)

Copie e cole o código abaixo no seu arquivo script.js. Este é o código "limpo", sem comentários, para facilitar a cópia.

script.js
const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  { nome: "Borboleta", info: "Lepidoptera" },
  { nome: "Mosquito", info: "Diptera" }
];

let cartas = [];

pares.forEach(par => {
  cartas.push({ texto: par.nome, id: par.nome });
  cartas.push({ texto: par.info, id: par.nome });
});

cartas = cartas.sort(() => 0.5 - Math.random());

const tabuleiro = document.getElementById('game-board');
let cartasViradas = [];
let cartasCorretas = [];

function criarTabuleiro() {
  cartas.forEach((carta, indice) => {
    const div = document.createElement('div');
    div.classList.add('card');
    div.setAttribute('data-id', carta.id);
    div.setAttribute('data-indice', indice);
    div.innerText = "?";
    div.addEventListener('click', virarCarta);
    tabuleiro.appendChild(div);
  });
}

function virarCarta() {
  if (cartasViradas.length === 2) return;

  const cartaClicada = this;
  const indice = cartaClicada.getAttribute('data-indice');
  const dados = cartas[indice]; 

  if (cartaClicada.classList.contains('flipped') || cartasCorretas.includes(indice)) {
    return;
  }

  cartaClicada.innerText = dados.texto;
  cartaClicada.classList.add('flipped');
  
  cartasViradas.push({ indice: indice, id: dados.id }); 

  if (cartasViradas.length === 2) {
    verificarPar();
  }
}

function verificarPar() {
  const [carta1, carta2] = cartasViradas;
  const status = document.getElementById('status');

  if (carta1.id === carta2.id && carta1.indice !== carta2.indice) {
    status.innerText = "Você acertou um par!";
    cartasCorretas.push(carta1.indice, carta2.indice);
  } else {
    status.innerText = "Tente novamente!";
    
    setTimeout(() => {
      const elementos = document.querySelectorAll('.card');
      
      const el1 = elementos[carta1.indice];
      const el2 = elementos[carta2.indice];
      
      el1.innerText = "?";
      el2.innerText = "?";
      el1.classList.remove('flipped');
      el2.classList.remove('flipped');
      
    }, 800);
  }

  cartasViradas = [];

  if (cartasCorretas.length === cartas.length) {
    status.innerText = "Parabéns! Você completou o jogo!";
  }
}

criarTabuleiro();

3. Entendendo o Código JavaScript

O JavaScript (script.js) é o "cérebro" que controla toda a lógica: embaralhar, virar cartas, verificar pares e terminar o jogo. Vamos analisar o que colamos, bloco por bloco.

Bloco 1: Os Dados e Variáveis Iniciais

  • const pares = [...]: Esta é a lista de pares. É A ÚNICA COISA que você precisa mudar se quiser personalizar o jogo. Cada objeto { } tem um nome (ex: "Abelha") e uma informação (ex: "Hymenoptera").
  • let cartas = [];: Criamos um array (uma lista) vazio que vai guardar TODAS as cartas do jogo (8, no nosso caso).
  • pares.forEach(...): Usamos um loop para passar por cada item da nossa lista pares. Para cada par, ele "empurra" (push) duas novas cartas para o array cartas:
    1. Uma carta com o texto do nome (ex: "Abelha")
    2. Uma carta com o texto da informação (ex: "Hymenoptera")
    Note que ambas recebem o mesmo id: par.nome. É assim que saberemos que elas formam um par!
  • cartas = cartas.sort(...): Esta linha reembaralha o array cartas usando um truque matemático (0.5 - Math.random()) para torná-lo aleatório.
  • const tabuleiro = ...: Aqui, guardamos o elemento HTML #game-board em uma variável, para não termos que procurá-lo toda vez.
  • let cartasViradas = [];: Uma "memória de curto prazo". Guarda as 2 cartas que o jogador acabou de virar.
  • let cartasCorretas = [];: Uma "memória de longo prazo". Guarda todas as cartas que já formaram um par correto, para não desvirarem mais.

Bloco 2: A Função criarTabuleiro()

Isto é uma "Função" (uma receita). Ela não é executada até ser chamada. O trabalho dela é construir o tabuleiro:

  • cartas.forEach((carta, indice) => ...): Faz um loop por cada carta do nosso array embaralhado.
  • const div = document.createElement('div');: Cria um elemento <div> na memória do navegador.
  • div.classList.add('card');: Adiciona a classe .card a essa div, para que o CSS saiba como estilizá-la.
  • div.setAttribute(...): "Carimba" informações escondidas no HTML. Guardamos o data-id (ex: "Abelha") e o data-indice (a posição no array: 0, 1, 2, ...). Isso será crucial para checarmos os pares.
  • div.innerText = "?";: Coloca o texto inicial "?" no cartão.
  • div.addEventListener('click', virarCarta);: "Escuta" por um clique. Se o cartão for clicado, ele chama a função virarCarta.
  • tabuleiro.appendChild(div);: Coloca a div (o cartão) que estava na memória dentro do #game-board na página real.

Bloco 3: A Função virarCarta()

Esta função é chamada TODA VEZ que um cartão é clicado.

  • if (cartasViradas.length === 2) return;: Uma trava de segurança. Se o jogador já virou 2 cartas, ele não pode virar uma terceira. O return para a função imediatamente.
  • const cartaClicada = this;: A palavra-chave this se refere ao elemento exato que foi clicado (o div do cartão).
  • const indice = ...: Pega de volta o data-indice que guardamos no HTML.
  • if (cartaClicada.classList.contains('flipped') ...): Outra trava. Se a carta clicada já tem a classe .flipped (já está virada) ou se seu índice já está na lista de cartasCorretas, a função para.
  • cartaClicada.innerText = dados.texto;: Troca o "?" pelo texto real da carta (ex: "Abelha").
  • cartaClicada.classList.add('flipped');: Adiciona a classe .flipped. O CSS vê isso e "vira" o cartão visualmente.
  • cartasViradas.push(...): Adiciona a carta que acabamos de virar na nossa "memória de curto prazo".
  • if (cartasViradas.length === 2) { ... }: Se esta foi a segunda carta virada, agora é hora de checar se o par está certo. Ele chama a função verificarPar().

Bloco 4: A Função verificarPar()

Chamada logo após a segunda carta ser virada.

  • const [carta1, carta2] = cartasViradas;: Pega as duas cartas da "memória de curto prazo".
  • const status = ...: Pega o elemento <p id="status"> para podermos escrever nele.
  • if (carta1.id === carta2.id ...): A grande checagem! Ele verifica se o id da carta 1 é igual ao id da carta 2.
    • SE FOR IGUAL (ACERTOU): Ele escreve "Você acertou um par!" e "empurra" os índices das duas cartas para a "memória de longo prazo" (cartasCorretas).
    • SE FOR DIFERENTE (ERROU): Ele escreve "Tente novamente!".
  • setTimeout(() => { ... }, 800);: (Isto só acontece se o jogador ERROU). O setTimeout cria um "atraso". Ele espera 800 milissegundos (0.8 segundos) antes de executar o código dentro dele. Isso dá tempo para o jogador ver a segunda carta antes que ela desvire.
    • O código lá dentro simplesmente encontra os dois elementos de carta e remove a classe .flipped e restaura o texto "?".
  • cartasViradas = [];: Esvazia a "memória de curto prazo" para preparar para a próxima jogada.
  • if (cartasCorretas.length === cartas.length) { ... }: A checagem de vitória! Se o número de cartas corretas for igual ao número total de cartas, o jogador venceu.

Bloco 5: Iniciar o Jogo

  • criarTabuleiro();: Esta é a única linha de código que é executada "solta". Depois de definir todas as nossas receitas (funções), esta linha finalmente chama a função criarTabuleiro e dá início ao jogo.

Parte 4: Como Testar, Personalizar e Publicar

Como Testar

  1. Verifique se você salvou os três arquivos (index.html, style.css, script.js) na mesma pasta jogo-da-memoria.
  2. Dê um clique duplo no arquivo index.html.
  3. Ele vai abrir no seu navegador e o jogo deve funcionar!

Como Personalizar (Adicionar Mais Cartas)

Esta é a parte mais legal! Você pode facilmente mudar o tema do jogo.

Abra o arquivo script.js e vá até o topo, no "Bloco 1".

Encontre esta variável:

const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  ...
];

Para adicionar mais pares, basta adicionar novas linhas dentro dos colchetes [ ], seguindo o mesmo modelo. Por exemplo, para adicionar "Formiga":

const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  { nome: "Borboleta", info: "Lepidoptera" },
  { nome: "Mosquito", info: "Diptera" },
  { nome: "Formiga", info: "Hymenoptera" },
  { nome: "Besouro", info: "Coleoptera" }
];

O JavaScript cuida automaticamente de criar, embaralhar e ajustar o tabuleiro. (Atenção: o layout do CSS está para 4 colunas, então adicione pares em múltiplos de 2 para ficar bonito).

Como Publicar no GitHub Pages

Quer mostrar seu jogo para o mundo? É grátis e fácil:

  1. Crie um novo repositório no GitHub.
  2. Envie (faça "upload") seus três arquivos (index.html, style.css, script.js) para esse repositório.
  3. No seu repositório, vá em Settings (Configurações) > Pages.
  4. Em "Source", selecione a branch main (ou master) e clique em "Save".
  5. Depois de alguns segundos, o GitHub mostrará um link verde. Esse é o link do seu jogo online!

Parabéns! Você acabou de construir um jogo da web completo do zero!
