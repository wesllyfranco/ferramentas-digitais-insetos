Tutorial: Crie seu Próprio Jogo da Memória - Insetos

Olá, futuro programador! Bem-vindo ao seu segundo projeto.

Vamos construir, passo a passo, um "Jogo da Memória" com o tema de insetos. O objetivo é encontrar os pares corretos, que neste caso são o nome popular do inseto e sua classificação científica (Ordem).

Este tutorial foi feito para quem está começando, então vamos explicar cada detalhe!

Parte 0: As Ferramentas e a Preparação

Assim como no projeto anterior, precisamos organizar nossas ferramentas e nosso "terreno".

1. A Ferramenta (O Editor de Texto)

  • Vamos usar o Visual Studio Code (VS Code) novamente.

2. O Terreno (A Pasta do Projeto)

  • Vamos criar uma nova pasta para este projeto. Não misture com os arquivos do "jogo-flashcard".
  • Ação:
    1. Vá para sua Área de Trabalho ou "Documentos".
    2. Clique com o botão direito do mouse e escolha Novo > Pasta.
    3. Dê a esta pasta o nome de jogo-da-memoria.
    4. Abra o Visual Studio Code, vá em File (Arquivo) > Open Folder... (Abrir Pasta...) e escolha a pasta jogo-da-memoria.

Tudo pronto. Vamos criar 3 arquivos dentro desta pasta.

Parte 1: O Esqueleto do Jogo (HTML)

O HTML é a estrutura, os "ossos" da nossa página. Para este jogo, ele é bem simples. Precisamos apenas de três coisas:

  1. Um título (<h1>).
  2. Uma "caixa" vazia para o nosso tabuleiro (<div id="game-board">). O JavaScript vai preencher essa caixa com os cartões.
  3. Um parágrafo para mostrar mensagens como "Você acertou!" (<p id="status">).

1. Crie o arquivo:

  • Dentro do VS Code, na pasta jogo-da-memoria, crie um novo arquivo.
  • Dê o nome de index.html.

2. Escreva o código HTML:

Copie e cole o código abaixo no seu arquivo index.html.

index.html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Jogo da Memória - Insetos</title>
  
  <!-- 1. Ligando o arquivo de estilos (CSS) -->
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <h1>Jogo da Memória: Insetos</h1>
  
  <!-- 2. O tabuleiro vazio. O JavaScript vai criar os cartões aqui -->
  <div id="game-board"></div>
  
  <!-- 3. Onde as mensagens de status (Acertou/Errou) vão aparecer -->
  <p id="status"></p>

  <!-- 4. Ligando o arquivo de lógica (JavaScript) - IMPORTANTE ser no final -->
  <script src="script.js"></script>
</body>
</html>

Parte 2: A Aparência do Jogo (CSS)

O CSS cuida da "decoração" (cores, tamanhos, layout). Aqui, a parte mais importante é o estilo do .card e o que acontece quando ele "vira" (a classe .flipped).

1. Crie o arquivo:

  • Na mesma pasta, crie um novo arquivo.
  • Dê o nome de style.css.

2. Escreva o código CSS:

Copie e cole o código abaixo no seu arquivo style.css.

style.css
body {
  font-family: Arial, sans-serif;
  text-align: center;
  background-color: #f5f5f5;
}
h1 {
  margin-top: 20px;
  color: #333;
}

/* --- O Tabuleiro --- */
#game-board {
  display: grid; /* Usamos "grid" para organizar os cartões em colunas */
  
  /* Cria 4 colunas, cada uma com 120px de largura */
  grid-template-columns: repeat(4, 120px); 
  
  gap: 15px; /* Espaço entre os cartões */
  justify-content: center; /* Centraliza o grid na página */
  margin: 30px auto;
  max-width: 600px;
}

/* --- O Cartão (Verso) --- */
.card {
  width: 120px;
  height: 120px;
  background-color: #1c6dd0; /* Cor azul do "verso" do cartão */
  border-radius: 8px;
  cursor: pointer;
  color: white; /* Cor do texto ("?") */
  
  /* Isso centraliza o texto "?" dentro do cartão */
  display: flex;
  align-items: center;
  justify-content: center;
  
  font-size: 14px;
  font-weight: bold;
}

/* --- O Cartão "Virado" (Frente) --- */
/* Esta é a "mágica" do CSS. 
   Quando um cartão tiver a classe "flipped" (que o JavaScript vai adicionar), 
   ele vai mudar sua aparência para esta:
*/
.card.flipped {
  background-color: #fff; /* Fundo branco */
  color: #000; /* Texto preto */
  border: 2px solid #1c6dd0; /* Borda azul */
}

Parte 3: O Cérebro do Jogo (JavaScript)

O JavaScript (script.js) é o "cérebro" que controla toda a lógica: embaralhar, virar cartas, verificar pares e terminar o jogo.

Este arquivo é o mais complexo, então vamos olhar por partes (embora você vá colar tudo de uma vez).

1. Crie o arquivo:

  • Na mesma pasta, crie o último arquivo.
  • Dê o nome de script.js.

2. Escreva o código JavaScript:

Copie e cole o código abaixo no seu arquivo script.js.

script.js
/* --- Bloco 1: Os Dados e Variáveis Iniciais --- */

// Esta é a lista de pares. É A ÚNICA COISA que você precisa
// mudar se quiser personalizar o jogo (mais insetos, animais, etc.)
const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  { nome: "Borboleta", info: "Lepidoptera" },
  { nome: "Mosquito", info: "Diptera" }
  // Adicione mais pares aqui!
];

let cartas = []; // Array vazio que vai guardar TODAS as 8 cartas

// Vamos duplicar os pares e criar o array de "cartas"
pares.forEach(par => {
  // Adiciona a carta "nome" (ex: "Abelha")
  cartas.push({ texto: par.nome, id: par.nome }); 
  // Adiciona a carta "info" (ex: "Hymenoptera")
  cartas.push({ texto: par.info, id: par.nome }); // Note que o 'id' é o mesmo!
});

// Embaralha as cartas. Este é um truque comum para embaralhar arrays.
cartas = cartas.sort(() => 0.5 - Math.random());

// "Variáveis de memória" do jogo
const tabuleiro = document.getElementById('game-board');
let cartasViradas = [];   // Guarda as 2 cartas que estão viradas no momento
let cartasCorretas = [];  // Guarda os índices das cartas que já formaram par


/* --- Bloco 2: A "Fábrica" de Cartões --- */

// Esta função vai construir o tabuleiro
function criarTabuleiro() {
  // Para cada item no nosso array de "cartas" embaralhado...
  cartas.forEach((carta, indice) => {
    // 1. Crie um elemento 
 na memória
    const div = document.createElement('div');
    
    // 2. Adicione a classe 'card' (para o CSS estilizar)
    div.classList.add('card');
    
    // 3. Adicione "atributos de dados"
    // O 'data-id' guarda o ID do par (ex: "Abelha")
    div.setAttribute('data-id', carta.id);
    // O 'data-indice' guarda a posição da carta no array (0, 1, 2, ...)
    div.setAttribute('data-indice', indice);
    
    // 4. Coloque o texto inicial "?"
    div.innerText = "?";
    
    // 5. Adicione um "ouvinte" de clique.
    // Quando este 
 for clicado, ele vai chamar a função "virarCarta"
    div.addEventListener('click', virarCarta);
    
    // 6. Coloque o 
 pronto dentro do tabuleiro na página
    tabuleiro.appendChild(div);
  });
}

/* --- Bloco 3: A Ação de Virar a Carta --- */

// Esta função é chamada TODA VEZ que um cartão é clicado
function virarCarta() {
  // Se já temos 2 cartas viradas, não faça nada (espere elas serem checadas)
  if (cartasViradas.length === 2) return;

  const cartaClicada = this; // 'this' se refere ao 
 que foi clicado
  
  // Pega o índice que guardamos no 'data-indice'
  const indice = cartaClicada.getAttribute('data-indice'); 
  
  // Pega os dados completos da carta (texto e id)
  const dados = cartas[indice]; 

  // Se a carta já está virada (flipped) ou já faz parte de um par (corretas),
  // não faça nada.
  if (cartaClicada.classList.contains('flipped') || cartasCorretas.includes(indice)) {
    return;
  }

  // A MÁGICA ACONTECE AQUI:
  // 1. Mostra o texto da carta (ex: "Abelha")
  cartaClicada.innerText = dados.texto; 
  // 2. Adiciona a classe 'flipped' (para o CSS mudar a cor)
  cartaClicada.classList.add('flipped'); 
  
  // 3. Guarda a carta virada na nossa "memória de curto prazo"
  cartasViradas.push({ indice: indice, id: dados.id }); 

  // 4. Se esta é a SEGUNDA carta virada...
  if (cartasViradas.length === 2) {
    // ...chame a função que verifica se o par está certo.
    verificarPar();
  }
}

/* --- Bloco 4: A Verificação do Par --- */

function verificarPar() {
  const [carta1, carta2] = cartasViradas; // Pega as duas cartas da memória
  const status = document.getElementById('status'); // Onde vamos escrever as mensagens

  // Checa se o 'id' das duas é o mesmo (ex: "Abelha" === "Abelha")
  // E também se não clicamos na *mesma carta* duas vezes
  if (carta1.id === carta2.id && carta1.indice !== carta2.indice) {
    // É UM PAR!
    status.innerText = "Você acertou um par!";
    // Adicionamos os índices na nossa "memória de longo prazo"
    cartasCorretas.push(carta1.indice, carta2.indice);
    
  } else {
    // NÃO É UM PAR!
    status.innerText = "Tente novamente!";
    
    // Aqui usamos 'setTimeout' para dar um tempinho (800ms)
    // para o jogador VER a segunda carta antes de desvirar.
    setTimeout(() => {
      const elementos = document.querySelectorAll('.card'); // Pega todos os cartões
      
      // Pega os elementos DOM das duas cartas que erramos
      const el1 = elementos[carta1.indice];
      const el2 = elementos[carta2.indice];
      
      // Desvira as duas
      el1.innerText = "?";
      el2.innerText = "?";
      el1.classList.remove('flipped');
      el2.classList.remove('flipped');
      
    }, 800); // 800 milissegundos = 0.8 segundos
  }

  // Limpa a "memória de curto prazo" para a próxima jogada
  cartasViradas = [];

  // Checagem de Vitória:
  // Se o número de cartas corretas é igual ao total de cartas...
  if (cartasCorretas.length === cartas.length) {
    status.innerText = "Parabéns! Você completou o jogo!";
  }
}

/* --- Bloco 5: Iniciar o Jogo --- */

// Finalmente, chamamos a função "fábrica" para construir o jogo
// assim que a página carrega.
criarTabuleiro();

Parte 4: Como Testar, Personalizar e Publicar

Como Testar

  1. Verifique se você salvou os três arquivos (index.html, style.css, script.js) na mesma pasta jogo-da-memoria.
  2. Dê um clique duplo no arquivo index.html.
  3. Ele vai abrir no seu navegador e o jogo deve funcionar!

Como Personalizar (Adicionar Mais Cartas)

Esta é a parte mais legal! Você pode facilmente mudar o tema do jogo.

Abra o arquivo script.js e vá até o topo, no "Bloco 1".

Encontre esta variável:

const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  ...
];

Para adicionar mais pares, basta adicionar novas linhas dentro dos colchetes [ ], seguindo o mesmo modelo. Por exemplo, para adicionar "Formiga":

const pares = [
  { nome: "Abelha", info: "Hymenoptera" },
  { nome: "Joaninha", info: "Coleoptera" },
  { nome: "Borboleta", info: "Lepidoptera" },
  { nome: "Mosquito", info: "Diptera" },
  { nome: "Formiga", info: "Hymenoptera" },
  { nome: "Besouro", info: "Coleoptera" }
];

O JavaScript cuida automaticamente de criar, embaralhar e ajustar o tabuleiro. (Atenção: o layout do CSS está para 4 colunas, então adicione pares em múltiplos de 2 para ficar bonito).

Como Publicar no GitHub Pages

Quer mostrar seu jogo para o mundo? É grátis e fácil:

  1. Crie um novo repositório no GitHub.
  2. Envie (faça "upload") seus três arquivos (index.html, style.css, script.js) para esse repositório.
  3. No seu repositório, vá em Settings (Configurações) > Pages.
  4. Em "Source", selecione a branch main (ou master) e clique em "Save".
  5. Depois de alguns segundos, o GitHub mostrará um link verde. Esse é o link do seu jogo online!
