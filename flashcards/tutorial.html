Tutorial: Como Criar seu Próprio Jogo de Cartas (FlashCard)

Olá, futuro programador! Bem-vindo ao seu primeiro projeto.

Vamos construir, passo a passo, um "Jogo de FlashCards". É um jogo onde você vê uma pergunta em um cartão e, ao clicar nele, o cartão vira para revelar a resposta. É muito usado para estudar!

Não se preocupe se você nunca escreveu uma linha de código. Vamos fazer tudo juntos, do começo ao fim.

Parte 0: As Ferramentas e a Preparação

Antes de construir uma casa, você precisa de ferramentas e de um terreno. Na programação, é a mesma coisa.

1. A Ferramenta (O Editor de Texto)

  • Precisamos de um programa para escrever nosso código.
  • Recomendamos um programa gratuito chamado Visual Studio Code (ou "VS Code"). Ele é o mais usado no mundo, é fácil de usar e ajuda você colorindo o código.
  • Ação: Peça ajuda para baixar e instalar o "Visual Studio Code" no seu computador. Você pode baixar aqui. Há também uma versão online que pode ser acessada por esse link

2. O Terreno (A Pasta do Projeto)

  • Não podemos deixar nossos arquivos jogados na Área de Trabalho. Vamos criar uma "pasta" para guardar todos os arquivos do nosso jogo.
  • Ação:
    1. Vá para sua Área de Trabalho ou para a pasta "Documentos".
    2. Clique com o botão direito do mouse e escolha Novo > Pasta.
    3. Dê a esta pasta o nome de jogo-flashcard.
    4. Abra o Visual Studio Code. No menu de cima, clique em File (Arquivo) > Open Folder... (Abrir Pasta...) e escolha a pasta jogo-flashcard que você acabou de criar.

Agora, estamos prontos! Toda a mágica vai acontecer dentro dessa pasta.

Parte 1: O Esqueleto do Jogo (HTML)

O primeiro arquivo que vamos criar é o HTML. Pense no HTML como os ossos do nosso site. Ele diz "aqui vai ter um título", "aqui vai ser a área dos cartões", "aqui vai ser o rodapé".

1. Crie o arquivo:

  • Dentro do VS Code, na lateral esquerda (onde está o nome jogo-flashcard), clique no ícone de "Novo Arquivo" (parece uma folha de papel).
  • Dê o nome de index.html e aperte Enter. É muito importante que termine com .html.

2. Escreva o código HTML:

Copie e cole o código abaixo exatamente como está dentro do seu arquivo index.html.

index.html
<!DOCTYPE html>
<html lang="pt-br">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- 1. Aqui estamos "linkando" nosso arquivo de estilo (que vamos criar) -->
    <link rel="stylesheet" href="style.css">
    
    <title>FlashCard</title>
</head>

<body>
    <main>
        <!-- 2. Esta é a "caixa" principal onde nossos cartões vão aparecer -->
        <section id="container">
          <!-- Os cartões vão ser criados pelo JavaScript aqui dentro -->
        </section>
    </main>
    
    <footer>
        <p>Projeto desenvolvido por Aluno - 2B </p>
    </footer>

    <!-- 3. Aqui estamos "chamando" nossos arquivos de cérebro (JavaScript) -->
    <!-- A ordem é IMPORTANTE! Primeiro a "fábrica" (app.js), depois as "perguntas" (perguntas.js) -->
    <script src="app.js"></script>
    <script src="perguntas.js"></script>
</body>

</html>

Parte 2: A Aparência do Jogo (CSS)

Nosso jogo ainda parece feio, só texto. Precisamos de CSS! Pense no CSS como as roupas, a pintura e a decoração do nosso site. Ele cuida das cores, dos tamanhos e das animações.

1. Crie o arquivo:

  • Assim como antes, crie um novo arquivo na sua pasta jogo-flashcard.
  • Dê o nome de style.css e aperte Enter.

2. Escreva o código CSS:

Copie e cole o código abaixo dentro do seu arquivo style.css.

style.css
:root {
    /* Criamos "apelidos" para nossas cores, para ficar fácil de mudar depois */
    --text-color: #dbe4ef;
    --card-front-color: #144480;
    --card-back-color: #0074bf;
}

body {
    background: url('img/bg-desktop.webp'); /* Você pode trocar isso por uma cor, ex: background-color: #f0f0f0; */
    font-family: 'Bai Jamjuree'; /* Tipo da letra */
}

#container {
    display: flex; /* Coloca os cartões um ao lado do outro */
    flex-wrap: wrap; /* Se não couber, joga para a linha de baixo */
    justify-content: space-between; /* Espaçamento bonito */
    padding: 4rem;
    gap: 3rem; /* Espaço entre os cartões */
}

.cartao {
    margin: 1rem 1rem;
    height: 20rem; /* Altura do cartão */
    flex-grow: 1;
    flex-basis: calc(33% - 6rem); /* Tenta colocar 3 cartões por linha */
}

.cartao_conteudo {
    background-color: var(--card-front-color); /* Cor da frente */
    text-align: center;
    height: 100%;
    
    /* --- A MÁGICA 3D COMEÇA AQUI --- */
    transform-style: preserve-3d; /* Avisa que vai ter coisa 3D aqui dentro */
    transition: transform 300ms ease-in-out; /* A animação de "virar" */
}

.cartao_conteudo h3 {
    color: var(--text-color);
    border: 1px solid var(--text-color);
    text-align: left;
    padding: 0.5rem;
    position: absolute; /* Posição "flutuante" */
    margin: 0.6rem;
    border-radius: 0.6rem;
    font-size: 1vw;
    backface-visibility: hidden; /* Esconde o que está "atrás" do cartão */
}

.cartao_conteudo p {
    margin-top: 1rem;
    padding: 2rem;
    margin-top: 4rem;
    font-size: 1.4vw;
}

.cartao_conteudo_pergunta p {
    color: var(--text-color);
    font-weight: 500;
}

.cartao_conteudo_resposta p {
    color: var(--card-back-color);
    font-weight: 700;
}


/* --- A MÁGICA 3D (CONTINUAÇÃO) --- */

/* Esta é a regra que "vira" o cartão quando o JavaScript adiciona a classe "active" */
.cartao.active .cartao_conteudo {
    transform: rotateY(180deg);
}

.cartao_conteudo_pergunta,
.cartao_conteudo_resposta {
    backface-visibility: hidden; /* Esconde o "verso" de cada lado */
    position: absolute; /* Coloca um em cima do outro */
    height: 100%;
    width: 100%;
    box-sizing: border-box;
}

.cartao_conteudo_resposta {
    transform: rotateY(180deg); /* A resposta já começa "virada" */
    background-color: rgba(0, 244, 191, 0.2);
    border: 4px solid var(--card-back-color);
}

/* --- ESTILOS DO RODAPÉ --- */
footer {
    background-color: black;
    color: white;
    position: fixed; /* Deixa o rodapé fixo lá embaixo */
    bottom: 0;
    width: 100%;
    heigth: 2rem;
}

footer p {
    text-align: center;
    font-size: 0.6rem;
    margin-top: 0.5rem;
}

/* --- AJUSTES PARA CELULAR --- */
@media (max-width: 560px) {
    body {
        background: url('img/bg-mobile.webp');
    }

    .cartao {
        flex: 1 0 calc(100% - 1rem); /* Faz o cartão ocupar 100% da largura no celular */
    }

    .cartao_conteudo h3 {
        font-size: 3vw;
    }

    .cartao_conteudo p {
        font-size: 3.8vw
    }
}

Observação: O background usa imagens (img/bg-desktop.webp) que não temos. O fundo pode ficar em branco, e isso é normal! Se quiser, troque as linhas background: url(...) por background-color: #f0f0f0; para ter um fundo cinza claro.

Parte 3: O Cérebro do Jogo (JavaScript - app.js)

Até agora, só temos uma página bonita, mas sem vida. O JavaScript é o cérebro do nosso jogo. Ele que faz as coisas acontecerem.

O app.js será nossa "Fábrica de Cartões". Ele contém a receita (a "função") de como construir um cartão.

1. Crie o arquivo:

  • Crie um novo arquivo na sua pasta jogo-flashcard.
  • Dê o nome de app.js e aperte Enter.

2. Escreva o código JavaScript:

Copie e cole o código abaixo dentro do seu arquivo app.js.

app.js
/* Isto é uma "Função".
  Pense nela como uma "receita de bolo" chamada "criaCartao".
  Para ela funcionar, ela precisa de 3 ingredientes:
  1. categoria
  2. pergunta
  3. resposta
*/
function criaCartao(categoria, pergunta, resposta) {
  
  // 1. Encontra a "caixa container" que fizemos no HTML
  let container = document.getElementById('container');

  // 2. Cria um novo elemento <article> (o cartão) na memória
  let cartao = document.createElement('article');
  
  // 3. Dá ao cartão uma "etiqueta" (classe) chamada "cartao", para o CSS saber estilizá-lo
  cartao.className = 'cartao';


  // 4. Aqui colocamos todo o HTML da "pergunta" e "resposta" dentro do cartão
  cartao.innerHTML = `
   <div class="cartao_conteudo">
                    <h3>${categoria}</h3>
                    <div class="cartao_conteudo_pergunta">
                        <p>${pergunta}</p>
                    </div>
                    <div class="cartao_conteudo_resposta">
                        <p>${resposta}</p>
                    </div>
                </div>
                `
  
  // 5. Agora, colocamos um "ouvinte" no cartão
  //    É como dizer: "Ei, cartão, fique esperando por um clique!"
  cartao.addEventListener('click', function() {
    
    // 6. Quando o clique acontecer, adicione ou remova a classe "active"
    //    É como um interruptor de luz. O CSS vai ver essa classe e vai "virar" o cartão.
    cartao.classList.toggle('active');
  });
  
  // 7. Por fim, pegamos o cartão que criamos na memória e o colocamos
  //    dentro do "container" na nossa página.
  container.appendChild(cartao);
}

Parte 4: As Perguntas (JavaScript - perguntas.js)

Nossa "fábrica" (app.js) está pronta. Mas ela não faz nada sozinha. Precisamos usar a fábrica!

O arquivo perguntas.js é a "linha de produção". É aqui que vamos chamar a receita criaCartao várias vezes, passando os ingredientes (as perguntas) que queremos.

1. Crie o arquivo:

  • Crie um novo arquivo na sua pasta jogo-flashcard.
  • Dê o nome de perguntas.js e aperte Enter.

2. Escreva o código JavaScript:

Copie e cole o código abaixo dentro do seu arquivo perguntas.js.

perguntas.js
// Aqui estamos "chamando" a receita (função) que fizemos no app.js

// Cartão 1
criaCartao(
    'Futebol',
    'Qual seleção tem mais copas do mundo?',
    'Brasil'
)

// Cartão 2
criaCartao(
    'Programação',
    'O que é HTML?',
    'Linguagem de Marcação'
)

// Cartão 3
criaCartao(
    'Programação',
    'O que é CSS?',
    'Linguagem de Estilo'
)

// Cartão 4
criaCartao(
    'Geografia',
    'Qual a capital da França?',
    'Paris'
)

// Cartão 5
criaCartao(
    'Ciências',
    'Qual o símbolo químico da água?',
    'H₂O'
)

Quer adicionar mais cartões?

É fácil! Basta copiar e colar o bloco abaixo no fim do arquivo perguntas.js e trocar o conteúdo:

criaCartao(
    'SUA CATEGORIA',
    'SUA PERGUNTA',
    'SUA RESPOSTA'
)

Parte 5: A Hora da Verdade! (Como Ver o Jogo)

Terminamos! Você criou 4 arquivos:

  1. index.html (o esqueleto)
  2. style.css (a aparência)
  3. app.js (a fábrica de cartões)
  4. perguntas.js (a lista de perguntas)

Para jogar:

  1. Abra a pasta jogo-flashcard no seu computador (fora do VS Code, no explorador de arquivos).
  2. Você verá os 4 arquivos que criou.
  3. Dê um clique duplo no arquivo index.html
  4. Ele vai abrir no seu navegador de internet (Google Chrome, Firefox, etc.) e... o seu jogo vai estar funcionando!

Clique nos cartões para ver a mágica acontecer.

Parabéns! Você acabou de construir um jogo da web completo do zero!
